# Hybrid Nonlinear/Linear Control of an Inverted Pendulum
## Project Overview
This repository contains a dual-mode hybrid control architecture for a classic underactuated Inverted Pendulum (Cart-Pole) system within MATLAB/Simulink, developed using AI-assisted development while analyzing the results to optimize transition boundary behavior. The system successfully executes a non-linear swing-up routine from a dead-hanging position and seamlessly transitions to an optimal linear regulator to maintain upright stability under physical actuator constraints.
## Key Technical Contributions & Implementation Milestones
- **First-Principles Rigid Body Dynamics:** Derived the explicit equations of motion for a two-degree-of-freedom cart-pendulum system using Euler-Lagrange mechanics. Modeled the plant natively in Simulink using atomic math blocks to prevent algebraic loops and ensure clean signal routing.
- **Nonlinear Energy-Shaping Swing-Up Control:** Developed a Lyapunov-based energy-shaping controller that commands pure feedforward acceleration. The control law dynamically pumps mechanical energy into the pendulum rod until the total system energy matches the potential energy apex of the upright equilibrium point.
- **State-Space Linear Quadratic Regulator (LQR):** Linearized the highly non-linear dynamics around the unstable upright equilibrium point ($[0,0,0,0]^T$). Used MATLAB to solve the Continuous-Time Algebraic Riccati Equation, designing an optimal feedback gain vector $\mathbf{K}$ that prioritizes strict cart positioning and rapid pole deflection rejection.
- **Four-Quadrant Trigonometric Signal Conditioning:** Implemented an atan2 ($\sin\theta / \cos\theta$) coordinate wrapping circuit. This maps infinite angular accumulation back to a bounded $[-\pi, \pi]$ domain, resolving a critical control failure mode by allowing the controller to recognize the shortest angular distance to the target during multi-turn rotations.
- **State-Machine Handoff & Actuator Saturation Safeguards:** Constructed a conditional switching logic engine that dynamically activates the LQR balancing loop only when the state enters a strict "Catch Window" ($\pm 15^\circ$ tilt angle and $\leq 1.5\,\text{rad/s}$ angular velocity). Integrated a global saturation block ($\pm 25\,\text{N}$) to accurately simulate real-world DC motor current limitations, verified via the Simulink Simulation Data Inspector.
- **Model Refactoring & Controller Implementation:** Refactored the foundational plant model originally developed during the commencement of sophomore undergraduate studies, upgrading the overall architecture to support the newly implemented dual-mode hybrid control loop.
## Repository Structure & Legacy Artifacts
- **`cart_pole.slx` & `init_params.m`:** The primary active production environment. Run `init_params.m` first to load the optimal LQR weight matrices ($\mathbf{Q}$, $\mathbf{R}$) and swing-up parameters into the workspace before executing the Simulink model.
- **Legacy Files (`pendulum.m`, `pendulum_2.slx`):** Retained purely for historical tracking. These represent the initial un-refactored rigid-body plant dynamics derived during early sophomore days prior to the integration of the hybrid control loops.
